### Front-End
##

**Criação do Projeto**.

- Criação da pasta frontend.
- Habilitar a execução de scripts no Windows.
- Prompt de comando, globalmente.
```
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```

**Criar o projeto**.
```    
npx create-next-app . --ts
``` 

**Iniciar o servidor**
```
npm run dev
```
